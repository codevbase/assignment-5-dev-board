Question-1: What are the different ways to select an element in the DOM?

Ans: There are multiple ways to select an element in the DOM. Some of the most common ways are:
  • getElementById
  • getElementsByClassName
  • getElementsByTagName
  • querySelector
  • querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans: innerHTML: It returns the HTML content of an element. It can also be used to set the HTML content of an element. innerText: It returns the text content of an element. It does not return the text content of hidden elements. textContent: It returns the text content of an element. It returns the text content of hidden elements as well.

Question-3: What is event delegation in the DOM?

Ans: Event delegation is a technique in which we attach a single event listener to a parent element that will fire for all children matching a selector, whether they are present now or in the future.

Question-4: What is event bubbling in the DOM?

Ans: Event bubbling is a mechanism in which an event propagates through the DOM hierarchy. When an event is fired on an element, it first runs the event handlers on it, then on its parent, then all the way up on other ancestors.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans: To create an element, we use the createElement method. To add an element, we use the appendChild method. To remove an element, we use the removeChild method.
